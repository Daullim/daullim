# ADR-010 시연 시군구 — 서울 관악구(도시) + 전북 임실군(농촌)

## 상태
승인 (2026-07-20, 결정자: 이윤서) — 조건부 교체 조항 포함

## 맥락
- 확정: 시연 대상 = 도시 자치구 1 + 농촌 군 1 (PRD v2 대안④·WBS D2). 도농 전환은 지역 선택 기반 자동 결정(기능명세서 FR-001).
- 도농판별 정의서 §6 가이드: 도시 = URBAN 비율 높은 서울 자치구, 농촌 = 절대 필터(비아파트≥80% ∧ 고령≥40%) 충족 많은 전북 군.
- seed 데이터 산출 범위·GeoJSON 클리핑(ADR-008 §9)과 직결.
- 한계: 후보의 데이터 검증은 seed 첫 실행(1주차) 후 가능 — 기준+1순위+교체 조건 구조로 확정.

## 검토한 선택지
- 도시: **관악구(채택)** / 중랑구·강북구(노후 다세대 밀집 — 차순위)
- 농촌: **임실군(채택)** / 순창군·진안군(차순위)
- 보류(seed 후 확정) — GeoJSON 클리핑 범위 확정 지연으로 기각

## 결정
- **도시형 시연: 서울 관악구.** 근거: 1인가구 비율 전국 최상위·다가구/고시원 밀집(도시 내 취약 서사), PoC 서울 데이터 커버리지 내.
- **농촌형 시연: 전북 임실군.** 근거: 고령화율 전북 최상위권 — 절대 필터 충족 개연성 최대, 소방 원거리 서사.
- **교체 조건:** seed 첫 실행에서 ① 관악구 URBAN 셀 화재 이력 밀도가 지도 시연에 빈약, 또는 ② 임실군 절대 필터 충족 가구가 큐 구성에 부족하면 차순위로 교체 제안(본 ADR 개정).

## 근거
- 두 지역의 대비(밀집 도시 취약 vs 초고령 농촌)가 도농 이원 알고리즘의 정체성 시연에 최대 효과.

## 결과 (트레이드오프 수용)
- 데이터 검증 전 확정에 따른 교체 가능성 수용 — 교체 시 GeoJSON 재생성만 필요(pipeline 파라미터화로 비용 최소화).
