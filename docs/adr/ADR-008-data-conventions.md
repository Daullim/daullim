# ADR-008 데이터 처리 컨벤션 — PoC 실측 함정의 코드 규칙화

## 상태
승인 (2026-07-20, 결정자: 이윤서)

## 맥락
- PoC(P1·P2)와 v4 §3에서 실측된 데이터 함정을 seed 파이프라인 필수 유틸로 명문화(NFR-05).
- 원칙: **함정 규칙은 문서가 아니라 코드에 산다** — 모든 스크립트는 공용 유틸 경유 필수, 직접 `pd.read_csv` 금지.

## 결정 — `pipeline/daullim_data/utils.py` 규칙 9종
(모듈명은 repo명 `daullim`과 일치 — ADR-001)
1. **인코딩**: `read_csv_kr()` — UTF-8-SIG → cp949 fallback, 자동 감지 금지. 쓰기 UTF-8(BOM 없음).
2. **한글 정규화**: 파일명·주소는 비교/조인 전 NFC 강제 (macOS NFD 함정).
3. **월 필드**: `normalize_month()` — 비패딩('1'~'9') → 2자리 패딩.
4. **값 범위 검증**: `validate_ranges()` — 습도 0~100·적설 ≥0 등 범위표, 위반 시 **fail-fast** + 필드 충전율 리포트 (습도·적설 스왑 실측 대응).
5. **시도 경계 클리핑**: `clip_to_sido()` — 존 프리픽스 화이트리스트(서울=다사/부산=마라·마마/전북=나마·다마·라마). 미클리핑 시 14배 팽창 실측.
6. **격자 조인**: `grid500_to_1km()` — 문자열 유도(다사46a41a→다사4641), 100% 일치 검증 근거.
7. **좌표계**: 저장·교환 EPSG:4326 / 계산 EPSG:5179. pyproj 명시 변환, 변수 접미사 `_4326`/`_5179`.
8. **연도 정규화**: 2021 라벨 드리프트 통제 — 시계열 가중 산출 유틸에 고정.
9. **산출 원자성**: `building_queue.csv` + GeoJSON을 한 실행에서 동시 생성. GeoJSON 좌표 5자리 반올림 + 시연 시군구 클리핑.

운영: 유틸에 pytest 함정 재현 테스트(D9 연계). `data/README.md`에 함정 요약 1페이지(온보딩 겸용).

## 근거
- 전 항목 PoC 실측 근거 보유 — 신규 리스크가 아니라 재발 방지 규칙화.

## 결과 (트레이드오프 수용)
- fail-fast로 파이프라인이 자주 멈출 수 있음 — 의도된 동작(무결성 > 편의)으로 수용.
